# Chat Box

A simple chat box

## Requirements
- go
- browser

## Backend

- Clone the repository

    `git clone git@github.com:claudesky/hello-go.git`


- Build the backend

    `cd hello-go`

    `go build .`

- Run the backend

    `./hello-go`

## Frontend

- Open the index html file

    `google-chrome-stable ./frontend/index.html`
